# AGENTS.md

Guidance for AI agents working in this repository.

## Agent skills

### Issue tracker

Issues and PRDs live as GitHub issues in `AaronAbuUsama/hosted-agents`, managed
with the `gh` CLI. External PRs are not a triage surface. See
`docs/agents/issue-tracker.md`.

### Triage labels

Canonical five-role vocabulary, each role mapped to its own name (`needs-triage`,
`needs-info`, `ready-for-agent`, `ready-for-human`, `wontfix`). See
`docs/agents/triage-labels.md`.

### Domain docs

Single-context: `CONTEXT.md` + `docs/adr/` at the repo root. See
`docs/agents/domain.md`.
